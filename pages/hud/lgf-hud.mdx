## Installation

### Prerequisites
- QBCore Framework (latest version)
- Basic understanding of FiveM resource management

### Download & Setup

1. **Download the Resource**
   ```bash
   # Download from GitHub releases or trusted source
   cd resources/[standalone]
   git clone https://github.com/example/lgf-hud.git
   ```

2. **Add to server.cfg**
   ```cfg
   # Add after qb-core
   start lgf-hud
   ```

3. **Restart Server**
   ```bash
   restart [server-name]
   # or
   refresh && start lgf-hud
   ```

### Dependencies
- `qb-core` - QBCore framework
- `qb-fuel` - For fuel gauge functionality (optional)
- `qb-stress` - For stress system integration (optional)

## Configuration

### Basic Configuration

The main configuration file is located at `config.lua`:

```lua
Config = {}

-- General Settings
Config.UpdateInterval = 100  -- Update frequency in ms
Config.EnableDebug = false   -- Enable debug mode

-- HUD Elements
Config.HUD = {
    -- Health Bar
    health = {
        enabled = true,
        position = { x = 120, y = 50 },
        size = { width = 200, height = 20 },
        color = '#ff4757',
        backgroundColor = 'rgba(0, 0, 0, 0.3)'
    },
    
    -- Armor Bar  
    armor = {
        enabled = true,
        position = { x = 120, y = 80 },
        size = { width = 200, height = 20 },
        color = '#3742fa',
        backgroundColor = 'rgba(0, 0, 0, 0.3)'
    },
    
    -- Hunger Bar
    hunger = {
        enabled = true,
        position = { x = 120, y = 110 },
        size = { width = 200, height = 20 },
        color = '#ff6b35',
        backgroundColor = 'rgba(0, 0, 0, 0.3)'
    },
    
    -- Thirst Bar
    thirst = {
        enabled = true,
        position = { x = 120, y = 140 },
        size = { width = 200, height = 20 },
        color = '#00d2d3',
        backgroundColor = 'rgba(0, 0, 0, 0.3)'
    }
}

-- Speedometer
Config.Speedometer = {
    enabled = true,
    unit = 'kmh', -- 'kmh' or 'mph'
    position = { x = -200, y = -100 },
    showGear = true,
    showFuel = true
}

-- Notifications
Config.Notifications = {
    position = 'top-right', -- 'top-right', 'top-left', 'bottom-right', 'bottom-left'
    duration = 5000, -- Duration in ms
    animations = true
}
```

#### Event Handlers

```lua
-- Register QBCore events
RegisterNetEvent('QBCore:Client:OnPlayerLoaded', function()
    TriggerEvent('lgf-hud:client:updatePlayerData')
end)

RegisterNetEvent('QBCore:Player:SetPlayerData', function(val)
    -- Update HUD with new player data
    local playerData = val
    TriggerEvent('lgf-hud:client:updateHUD', {
        health = GetEntityHealth(PlayerPedId()),
        armor = GetPedArmour(PlayerPedId()),
        hunger = playerData.metadata.hunger,
        thirst = playerData.metadata.thirst,
        stress = playerData.metadata.stress
    })
end)
```

## Commands and Usage

### Admin Commands

```lua
-- Toggle HUD visibility
/hud         -- Toggle entire HUD
/hud health  -- Toggle health bar only
/hud armor   -- Toggle armor bar only

-- Reset HUD position
/hudreset    -- Reset all elements to default positions

-- HUD configuration
/hudconfig   -- Open configuration menu (admin only)
```

### Player Commands


### Modifying HUD Elements

#### Adding Custom Elements

```lua
-- client/custom_elements.lua
RegisterNetEvent('lgf-hud:client:addCustomElement', function(elementData)
    -- Add custom element to HUD
    local element = {
        id = elementData.id,
        position = elementData.position,
        value = elementData.value,
        style = elementData.style
    }
    
    -- Register element with HUD system
    TriggerEvent('lgf-hud:registerElement', element)
end)
```

## Performance Optimization

### Best Practices

1. **Update Intervals**: Adjust update frequencies based on importance
   ```lua
   Config.UpdateIntervals = {
       health = 100,    -- Critical - fast updates
       hunger = 1000,   -- Less critical - slower updates
       speedometer = 50 -- Vehicle-dependent - very fast
   }
   ```

2. **Conditional Rendering**: Only show relevant elements
   ```lua
   Config.ConditionalDisplay = {
       speedometer = {
           onlyInVehicle = true,
           hideOnFoot = true
       },
       stress = {
           hideWhenLow = true,
           threshold = 10
       }
   }
   ```

3. **Resource Cleanup**: Properly clean up resources
   ```lua
   AddEventHandler('onResourceStop', function(resourceName)
       if resourceName == GetCurrentResourceName() then
           -- Cleanup HUD elements
           TriggerEvent('lgf-hud:client:cleanup')
       end
   end)
   ```

## Troubleshooting

### Common Issues

#### HUD Not Displaying

1. **Check Dependencies**: Ensure QBCore is running
   ```bash
   ensure qb-core
   ensure lgf-hud
   ```

2. **Verify Resource Order**: Make sure lgf-hud starts after qb-core
   ```cfg
   start qb-core
   start lgf-hud
   ```

3. **Check Console Errors**: Look for JavaScript/Lua errors
   ```bash
   # In server console
   con_filter_text "lgf-hud"
   ```

#### Performance Issues

1. **Reduce Update Frequency**: Increase update intervals
2. **Disable Unnecessary Elements**: Turn off unused HUD components
3. **Check for Conflicts**: Ensure no other HUD resources are conflicting

#### Configuration Not Saving

1. **Check File Permissions**: Ensure config files are writable
2. **Verify JSON Format**: Check for syntax errors in configuration
3. **Restart Resource**: Restart lgf-hud after configuration changes

## API Reference

### Client Events

```lua
-- Update HUD values
TriggerEvent('lgf-hud:client:updateHealth', health)
TriggerEvent('lgf-hud:client:updateArmor', armor)
TriggerEvent('lgf-hud:client:updateHunger', hunger)
TriggerEvent('lgf-hud:client:updateThirst', thirst)

-- Toggle HUD elements
TriggerEvent('lgf-hud:client:toggleHUD', visible)
TriggerEvent('lgf-hud:client:toggleElement', elementName, visible)

-- Configuration
TriggerEvent('lgf-hud:client:setTheme', themeName)
TriggerEvent('lgf-hud:client:setPosition', elementName, position)
```

### Server Events

```lua
-- Global HUD updates (admin only)
TriggerEvent('lgf-hud:server:updateGlobalConfig', config)
TriggerEvent('lgf-hud:server:broadcastUpdate', data)

-- Player-specific updates
TriggerClientEvent('lgf-hud:client:personalUpdate', source, data)
```

### Exports

```lua
-- Get HUD status
local isVisible = exports['lgf-hud']:IsHUDVisible()

-- Update specific elements
exports['lgf-hud']:UpdateHealth(newHealth)
exports['lgf-hud']:UpdateElement(elementName, value)

-- Configuration
exports['lgf-hud']:SetTheme(themeName)
exports['lgf-hud']:GetCurrentConfig()
```
