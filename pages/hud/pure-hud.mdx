## Installation

### System Requirements
- QBCore Framework (v1.0+)
- Modern browser engine (Chrome 80+, Firefox 75+)
- FiveM server with resource management permissions

### Installation Steps

1. **Download Pure HUD**
   ```bash
   cd resources/[qb]
   git clone https://github.com/example/pure-hud.git
   ```

2. **Install Dependencies**
   ```bash
   cd pure-hud
   npm install         # If using npm build process
   ```

3. **Server Configuration**
   ```cfg
   # Add to server.cfg after qb-core
   start pure-hud
   
   # Optional: Disable default QBCore HUD
   setr qb-hud:use_default false
   ```

4. **Permissions Setup**
   ```bash
   # Ensure proper file permissions
   chmod -R 755 resources/[qb]/pure-hud/
   ```

### Creating Custom Themes

1. **Create Theme File**
   ```css
   /* html/themes/mytheme.css */
   .pure-hud.theme-mytheme {
       --health-color: #your-color;
       --armor-color: #your-color;
       --hunger-color: #your-color;
       --thirst-color: #your-color;
       --stress-color: #your-color;
       
       /* Background styles */
       --bar-bg: rgba(0, 0, 0, 0.4);
       --bar-border: 1px solid rgba(255, 255, 255, 0.1);
       --bar-radius: 12px;
   }
   ```

2. **Register in Config**
   ```lua
   Config.AvailableThemes = {
       'default',
       'dark', 
       'neon',
       'minimal',
       'mytheme'  -- Your custom theme
   }
   ```

## Commands & Controls

### User Commands

```lua
/purehud                -- Toggle Pure HUD on/off
/purehud theme [name]   -- Change theme
/purehud scale [0.5-2]  -- Adjust HUD scale
/purehud reset          -- Reset to defaults
/purehud move           -- Enter position adjustment mode

-- Individual elements
/purehud toggle health  -- Toggle specific elements
/purehud toggle armor
/purehud toggle hunger
/purehud toggle thirst
```

### Position Adjustment Mode

When in adjustment mode:
- **Arrow Keys**: Fine adjustment (1px)
- **Shift + Arrows**: Coarse adjustment (10px)
- **Enter**: Save current position
- **ESC**: Cancel without saving
- **R**: Reset element to default position

### Admin Commands

```lua
/purehud admin          -- Open admin panel
/purehud config [player]-- Configure HUD for specific player  
/purehud reload         -- Reload configuration
/purehud debug          -- Toggle debug mode

-- Global settings
/purehud global theme   -- Set server-wide theme
/purehud global scale   -- Set server-wide scale
```

## API Integration

### QBCore Integration

```lua
-- Automatic QBCore event handling
RegisterNetEvent('QBCore:Client:OnPlayerLoaded', function()
    local PlayerData = QBCore.Functions.GetPlayerData()
    TriggerEvent('pure-hud:client:initialize', PlayerData)
end)

RegisterNetEvent('QBCore:Player:SetPlayerData', function(val)
    TriggerEvent('pure-hud:client:updatePlayerData', val)
end)

-- Status updates
RegisterNetEvent('hud:client:UpdateNeeds', function(hunger, thirst)
    TriggerEvent('pure-hud:client:updateHunger', hunger)
    TriggerEvent('pure-hud:client:updateThirst', thirst)
end)

RegisterNetEvent('hud:client:UpdateStress', function(stress)
    TriggerEvent('pure-hud:client:updateStress', stress)
end)
```

## Performance Optimization

### Efficient Updates

```lua
Config.OptimizedUpdates = {
    enabled = true,
    
    -- Update only when values change
    onChangeOnly = true,
    
    -- Batch updates to reduce calls
    batchUpdates = {
        enabled = true,
        interval = 100,             -- Batch every 100ms
        maxBatchSize = 10
    },
    
    -- Reduce updates when idle
    idleOptimization = {
        enabled = true,
        idleThreshold = 30000,      -- 30 seconds
        idleUpdateInterval = 1000   -- Update every 1s when idle
    }
}
```

### Memory Management

```lua
-- Automatic cleanup
CreateThread(function()
    while true do
        if Config.OptimizedUpdates.enabled then
            -- Clean up unused elements
            TriggerEvent('pure-hud:client:cleanup')
            
            -- Garbage collection
            collectgarbage('collect')
        end
        
        Wait(60000) -- Every minute
    end
end)
```

## Troubleshooting

### Common Issues

#### HUD Not Loading
1. Check resource order in server.cfg
2. Verify QBCore is running properly
3. Check console for JavaScript errors
4. Clear FiveM cache

#### Performance Issues
1. Increase update intervals
2. Disable unused elements
3. Use optimized update settings
4. Check for resource conflicts

#### Theme Not Applying
1. Verify theme exists in themes folder
2. Check CSS syntax errors
3. Restart resource after adding themes
4. Clear browser cache with Ctrl+F5
