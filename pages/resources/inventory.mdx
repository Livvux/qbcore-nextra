## Installation Guide

### Prerequisites
- QBCore Framework installed
- MySQL database access
- Basic understanding of FiveM resources

### General Installation Steps
1. Download the inventory system of your choice
2. Extract to your resources folder
3. Import any required SQL files
4. Configure the inventory in your server.cfg
5. Restart your server

### Configuration
Each inventory system has its own configuration file where you can customize:
- Weight limits
- Item restrictions
- UI appearance
- Shop configurations
- Storage options

## Choosing the Right Inventory

### For Beginners: QBCore Inventory
Perfect if you're just starting with QBCore and want a reliable, well-documented system.

### For Advanced Servers: LJ Inventory
Best choice for servers that want additional features and a modern UI.

### For Custom Setups: QB Inventory V2
Ideal for servers that need specific customizations and enhanced functionality.

## Common Issues & Solutions

### Inventory Not Opening
- Check if the resource is started in server.cfg
- Verify no other inventory resources are conflicting
- Check console for errors

### Items Not Showing
- Ensure item images are properly configured
- Check if items exist in your items.lua file
- Verify database connection

### Weight Issues
- Review weight configurations in config files
- Check if weight calculations are correct
- Verify player data is properly loaded
